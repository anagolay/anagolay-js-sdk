## Description

Wrapper of JSON.stringify().

## Links and Repo

## Install

```sh
# install latest version
yarn add @anagolay/op-json-enc

# or specific version
yarn add @anagolay/op-json-enc@0.3.1
```

## Usage

```ts
import jsonEnc from '@anagolay/op-json-enc'

const data = new U8intArray(7)
await jsonEnc(data)
```
