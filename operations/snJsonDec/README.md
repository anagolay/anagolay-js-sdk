## Install

```sh
# install latest version
yarn add @sensio/op-sn-json-dec

# or specific version
yarn add @sensio/op-sn-json-dec@0.3.1
```

## Usage

```ts
import snJsonDec from '@sensio/op-sn-json-dec'

const data = new U8intArray(7)
await snJsonDec(data)
```
